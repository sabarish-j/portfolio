# Portfolio

A personal portfolio website showcasing my projects, skills, experience, and contact information.

## Live Demo

Coming soon

## Overview

This portfolio is built to present my work, technical skills, and professional profile in a clean and interactive way. It includes sections for personal introduction, projects, skills, and contact details.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Three.js
- Framer Motion
- GSAP
- Lucide React

## Features

- Responsive design for desktop and mobile
- Interactive UI animations
- 3D visual elements
- Project showcase section
- Skills and tools section
- Contact information section
- Production-ready Vite build setup

## Getting Started

### Prerequisites

Make sure you have Node.js and npm installed.

Check your versions:

``bash
node --version
npm --version

**Installation**
**Clone the repository:**

git clone https://github.com/sabarish-j/portfolio.git
**Go to the project folder:**

cd portfolio
**Install dependencies:**

npm install
Start the development server:

npm run dev
Open the local URL shown in the terminal, usually:

http://localhost:5173
Available Scripts
Run the development server:

npm run dev
Build the project for production:

npm run build
Preview the production build locally:

npm run preview
Run linting:

npm run lint
Run TypeScript checks:

npm run typecheck

**Deployment**
This project can be deployed on Vercel, Netlify, or GitHub Pages.

Recommended deployment: Vercel

**Vercel Configuration**
Framework Preset: Vite
Build Command: npm run build
Output Directory: dist
Install Command: npm install
